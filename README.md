# 🌐 Lonely Legacy (LL) Official Web Platform
> **An Artist-Driven Subculture Platform & Media Archive**

---

## 1. 프로젝트 소개 및 목적 (Introduction & Purpose)
LL Crew의 활동 및 프로젝트에 확장해 아티스트 MD 판매 및 대외 활동을 비추고, 아티스트 브랜딩 강화를 목적으로 개발했습니다. 

보편적인 이커머스 시스템 구축을 넘어 아티스트 크루의 고유한 시각적·음악적 아카이브를 사용자에게 직관적으로 전달하는 인터랙티브 웹 플랫폼의 프로토타입입니다. 
서브컬처 브랜드인 Chrome Hearts 브랜드를 메인으로 IAB Studio, I4P등의 고감도 레이아웃과 UI 메커니즘(interaction 등)을 벤치마킹하여 LL 브랜드의 무드로 재해석했습니다.

---

## 2. 핵심 기능 및 기술적 특징 (Key Features)
* **크롬하츠 무드의 톤앤매너:** 다크 모드 지양, 정제된 밝은 베이지 톤의 고화질 백그라운드 텍스처 레이어 구현
* **CSS Scroll Snap 기반 마그네틱 스크롤:** 화면 전체를 제어하는 풀페이지 오버레이 구성을 통해 비주얼 몰입감 극대화
* **오버랩 사이드바 메뉴 (Chrome Hearts Style):** 메인 레이아웃의 밀림이나 왜곡 없이 좌측 엠블럼 오브젝트 클릭 시 오버랩되는 독립적 내비게이션 시스템
* **HTML5 Drag and Drop API 연동:** 상단 로고를 데스크탑 바탕화면으로 드래그 시 웹사이트 바로가기(.url 파일)가 자동 생성되는 피지컬 디바이스 인터랙션 구현

---

## 3. 개발 도구 및 환경 (Tech Stack)
* **Frontend Editor:** Windsurf (Cascade AI Assistant)
* **Languages:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Design & Mockup:** Adobe Photoshop / Illustrator
